# GroupWatch

## Description

GroupWatch is a real-time synchronized video streaming platform that allows groups of people to watch YouTube videos together from anywhere in the world. With built-in group chat functionality, users can communicate and share reactions while enjoying content together, creating a shared viewing experience despite physical distance.

## Key Features

- **Synchronized Playback** - All group members watch at the exact same timestamp
- **Group Chat** - Real-time messaging while watching
- **YouTube Integration** - Easy integration with YouTube links
- **Group Management** - Create and manage viewing groups
- **Live Reactions** - Share reactions in real-time
